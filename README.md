##<b>AI Powered Career Counsellor - SENSAI<b>

Sensai is a next-generation career optimization platform that leverages Google’s Gemini AI to provide job seekers with an unfair advantage. From real-time market insights to automated resume crafting, it's a complete ecosystem for career success.


##**​✨ Pro Features**

###**​📈 Industry Intelligence**
* ​Weekly Pulse: Automated background jobs (Inngest) fetch the latest market outlook, salary benchmarks, and trending skills.
* ​Data Visualization: Interactive charts powered by Recharts to visualize your career growth and market position.
###**​📝 AI Resume & Cover Letter**
* ​ATS Optimization: Built-in AI intelligence to make your resume machine-readable and impactful.
* ​One-Click Improvement: Transform weak job descriptions into powerful achievements using expert-level AI prompts.
* ​PDF Export: Download professionally formatted resumes directly from the browser using html2pdf.
###​**🎙️ Mock Interview Engine**
* ​Role-Specific Prep: Adaptive questioning based on your target industry and years of experience.
* ​Instant Feedback: Detailed performance analysis and specific "Actionable Improvement Tips" generated by AI.


##**​🛠️ Tech Stack**

* ​Framework: Next.js 15 (App Router)
* ​Styling: Tailwind CSS + Shadcn UI
* ​Database: PostgreSQL (Hosted on Neon DB)
* ​ORM: Prisma
* ​AI Engine: Google Gemini AI
* ​Authentication: Clerk
* ​Background Jobs: Inngest (Cron Jobs)
* ​Charts: Recharts
* ​Form Management: React Hook Form + Zod


##**🏁 Getting Started**
###​**Prerequisites:-**

1.​Node.js 15
2.​A Neon DB account (PostgreSQL)
​3.Clerk API Keys
4.​Google Gemini API Key


##**Installation**
1. Clone the repository:
 git clone https://github.com/sadhanamaurya3052005/AI-Powered-Career-Counsellor-SENSAI
2. Install Dependencies:
   npm install
3. Set up Environment Variables:
   DATABASE_URL=your_postgresql_url
   NEXT_PUBLIC_CLERK_PUBLISHABLE_KEY=your_clerk_pub_key
   CLERK_SECRET_KEY=your_clerk_secret_key
   GEMINI_API_KEY=your_gemini_api_key
4. Run Database Migrations:
   npx prisma generate
   npx prisma migrate dev
5. Start the Development Server:
   npm run dev
